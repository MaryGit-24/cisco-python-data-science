# Mondrian-art Project

## Dataset
Data Sources:

Images were downloaded from the website http://pietmondrian.rkdmonographs.nl (Catalogue Raisonné).

We wrote algorithms to extract the features from each painting.

## Objective
Extract features from each painting and perform analysis to answer specific questions.

## Methodology
- Data preprocessing  
- Feature extraction using Python  
- Analysis and visualizations  
- Code in `notebook.ipynb`
