# Cisco Python Data Science Projects

This repository contains all the projects I completed as part of the Cisco Python Data Science course. The projects span five modules, each focusing on different aspects of data analysis and machine learning. Each module has multiple projects with separate datasets and notebooks.

## Modules Overview

### 1. Coding for Answers
In this module, we practiced breaking down datasets to answer specific questions. The focus was on understanding the data and writing Python code to extract meaningful information.

### 2. Data Cleaning
This module focused on preparing datasets for analysis. It included tasks such as handling missing values, correcting data types, feature engineering, and preprocessing datasets to make them ready for modeling.

### 3. Data Modeling
In this module, we built predictive and analytical models to extract insights from data. This included regression, classification, and other modeling techniques to make data-driven predictions.

### 4. Data Storytelling
This module honed the skill of interpreting data and communicating insights effectively. The goal was to “listen” to the data and tell a meaningful story about what it reveals.

### 5. Data Visualization
In this module, we visualized data using Python libraries to make complex datasets understandable. Each project used different datasets to explore visualization techniques.

## Project Structure

Each module folder contains multiple projects. Inside each project folder, you will find:

- Python notebook(s) demonstrating the work  
- Dataset(s) used for the project  
- A project-level README detailing the dataset source, preprocessing steps, and methodology  

This structure allows each project to be **reproduced independently**.
